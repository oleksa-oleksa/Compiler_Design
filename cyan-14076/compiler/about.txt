  Cyan is a compiler toolkit that generates
  complete language processors in Java.
  It is based on Gentle (gentle.compilertools.net)
  and Accent (accent.compilertools.net).

  For documentation see cyan.compilertools.net.

  Go to directory 'example' for a first example.
  On Unix run 'generate.sh' to generate a language processor,
  run 'execute.sh' to execute the generated program.
  On Windows run 'generate.bat' and 'execute.bat'.
