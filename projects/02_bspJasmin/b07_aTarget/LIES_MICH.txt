Sinn und Moeglichkeiten dieses Ordners:

Angenommen, Sie haben mit einem alg-Compiler eine Jasmin-Datei
Target.j erzeugt. Dann kann man Target.j hier in eine Bytecode-Datei
Target.class uebersetzen und dann ausfuehren lassen. Dazu ist
allerdings immer auch das Laufzeitsystem (run time system) RTS.class
erforderlich.

Die einzelenen Schritte, um Target.j auszufuehren:

1. Die Datei Target.j in diesen Ordner bspJasmin\b06_aTarget kopieren.

2. Die Datei Target.j in eine Bytecode-Datei Target.class uebersetzen:
build Target.j

3. Die Datei RTS.java in eine Bytecode-Datei RTS.class uebersetzen:
build RTS.java

4. Target.class (zusammen mit RST.class) ausfuehren lassen:
java Target
